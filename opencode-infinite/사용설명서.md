# 🧠 클로드 4.5 팅킹 무한 매니저 (터미널 서브용)

**이 도구의 용도**:
평소에는 **Antigravity(오픈코드)**로 코딩하시다가, **리미트 걸릴 때만** 터미널에서 서브로 사용하세요.

---

## ⚡ 1초 순환 (가장 많이 쓸 명령어)

코딩하다가 멈추면 **터미널**을 열고 딱 이것만 치세요.

```powershell
ag rotate
```

👉 **효과**: 자동으로 다음 계정(`101` -> `102`...)으로 교체됩니다.
👉 **확인**: 터미널에 "전환되었습니다" 뜨면 다시 코딩 시작!

---

## 🌍 기타 명령어 (서브 기능)

필요할 때만 골라서 쓰세요.

- **`ag load`**
  : 계정 목록을 보고 직접 골라서 바꾸고 싶을 때.
- **`ag setup`**
  : 계정이 꼬이거나 새로 추가할 때 (초기 설정 다시 하기).
